## Requirements

1. Package located at `internal/module/module.go`
2. Define `Module` interface with methods:
   - `Name() string` - Return module name
   - `Description() string` - Return module description
   - `IsInstalled() (bool, error)` - Check if module is already installed
   - `Install(cfg *config.Config) error` - Install the module
3. Interface should be well-documented with godoc comments
4. Include example usage in comments
