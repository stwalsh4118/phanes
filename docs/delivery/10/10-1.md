## Requirements

1. Create `internal/runner/result.go` file
2. Define `ModuleStatus` type with constants:
   - `StatusInstalled` - Module was successfully installed
   - `StatusSkipped` - Module was skipped (already installed)
   - `StatusFailed` - Module installation failed
   - `StatusError` - Error occurred during module check/execution
3. Define `ModuleResult` struct with fields:
   - `Name` (string) - Module name
   - `Status` (ModuleStatus) - Execution status
   - `Error` (error) - Error details (only set if Status is Failed or Error)
   - `Duration` (time.Duration) - Optional execution time
